# Epic 2: 容器管理

> Phase: 1 (P0) + Phase 3 (P2)

---

## US-2.1 查看容器列表 `P0` `Phase 1`

**作为** 用户
**我想** 看到 VPS 上所有 Docker 容器的状态
**以便** 了解各服务运行情况

**验收标准:**
- [ ] 列表显示: 容器名、镜像、状态 (running/stopped/exited)、运行时长、CPU/内存占用
- [ ] 系统容器 (Speedtest/SWAG) 与应用容器分组显示
- [ ] 状态实时更新 (SSE)
- [ ] 支持按名称/状态过滤

**数据来源:**
- 独立模式: `GET /containers`
- 网关模式: `GET /api/v1/nodes/:id/containers`

**前端交互:**
```
┌────────────┬──────────────────┬──────────┬─────┬──────┬──────┐
│ Name       │ Image            │ Status   │ CPU │ MEM  │ Act. │
├────────────┼──────────────────┼──────────┼─────┼──────┼──────┤
│ ── System ─┼──────────────────┼──────────┼─────┼──────┼──────┤
│ speedtest  │ openspeedtest:.. │ 🟢 Up 5d │ 1%  │ 32M  │  ⋮  │
│ swag       │ linuxserver/s..  │ 🟢 Up 5d │ 0%  │ 45M  │  ⋮  │
│ ── Apps ───┼──────────────────┼──────────┼─────┼──────┼──────┤
│ wireguard  │ linuxserver/w..  │ 🟢 Up 3d │ 2%  │ 50M  │  ⋮  │
│ l2tp       │ hwdsl2/ipsec-..  │ 🔴 Exited│ 0%  │  0M  │  ⋮  │
└────────────┴──────────────────┴──────────┴─────┴──────┴──────┘
```

## US-2.2 容器操作 `P0` `Phase 1`

**作为** 用户
**我想** 启动/停止/重启/删除容器
**以便** 管理服务运行状态

**验收标准:**
- [ ] 每个容器提供操作菜单: Start / Stop / Restart / Delete
- [ ] 删除操作需二次确认
- [ ] 操作后实时更新容器状态 (乐观更新 + 服务端确认)
- [ ] 操作失败显示 Toast 错误信息 (如端口冲突、镜像缺失)
- [ ] 系统容器 (Speedtest/SWAG) 删除时显示警告 "这将影响基础服务"
- [ ] 操作进行中容器行显示 loading 状态

**前端交互:**
```
容器行右侧 ⋮ 按钮 → DropdownMenu:
  ▶ Start        (stopped 时)
  ⏸ Stop         (running 时)
  🔄 Restart      (running 时)
  ── separator ──
  🗑 Delete       (所有状态)

Delete 点击 → AlertDialog:
  "确定删除容器 wireguard?"
  ☐ 同时删除挂载卷
  [取消] [删除]
```

## US-2.3 查看容器日志 `P2` `Phase 3`

**作为** 用户
**我想** 查看容器的运行日志
**以便** 排查问题

**验收标准:**
- [ ] 点击容器打开日志面板 (Sheet 从右侧滑入)
- [ ] 默认显示最近 200 行
- [ ] 支持实时尾随 (tail -f 效果，通过 SSE)
- [ ] 支持按关键词搜索/高亮
- [ ] 支持下载完整日志
- [ ] 日志使用等宽字体 (monospace)，保留 ANSI 颜色

**前端交互:**
```
Sheet (width: 50%):
  ┌─────────────────────────────────────────────┐
  │  wireguard logs                        [×]  │
  │  [Search: ________] [⬇ Download] [Auto ☑]  │
  │─────────────────────────────────────────────│
  │  2026-03-12 10:00:01 [INFO] Started         │
  │  2026-03-12 10:00:02 [INFO] Peer connected  │
  │  2026-03-12 10:00:05 [WARN] Handshake ...   │
  │  ...                                        │
  │  (auto-scroll to bottom when Auto is on)    │
  └─────────────────────────────────────────────┘
```
